# MindID Site

## Language / Dil
- Always communicate in Turkish (Her zaman Turkce konusul)

## Tech Stack
- **Framework:** Next.js (App Router)
- **Language:** TypeScript
- **Styling:** Tailwind CSS
- **UI Components:** Radix UI
- **Package Manager:** npm

## Project Conventions
- Use `src/` directory for source code
- Follow Next.js App Router file conventions (`page.tsx`, `layout.tsx`, `loading.tsx`, etc.)
- Components go in `src/components/`
- Utility functions go in `src/lib/`
- TypeScript strict mode enabled

## Code Style
- Functional components with arrow functions
- Named exports preferred over default exports
- Use Tailwind utility classes directly; avoid custom CSS unless necessary
- Keep components small and focused

## Git
- Commit messages in English
- Use conventional commit format (`feat:`, `fix:`, `chore:`, etc.)
- Always run `git fetch origin` before starting any work
- After changes are approved by the user, push to remote with `git push`
